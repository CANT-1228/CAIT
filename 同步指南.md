# Heartopia攻略站 - 自动同步使用指南

## 快速开始

### 1. 安装Git（首次使用）

**下载地址：** https://git-scm.com/download/win

安装完成后，打开PowerShell运行：

```powershell
git config --global user.name "CANT-1228"
git config --global user.email "你的GitHub邮箱"
```

### 2. 初始化仓库（首次使用）

在项目目录运行以下命令：

```powershell
cd d:\21562\Documents\2104
git init
git remote add origin https://github.com/CANT-1228/CAIT.git
git add .
git commit -m "初始提交"
git push -u origin main
```

### 3. 日常使用流程

**每次修改代码后，只需运行：**

```powershell
.\auto-sync.ps1
```

或者在PowerShell中直接运行：

```powershell
git add .
git commit -m "更新内容"
git push
```

## 工作流程

```
本地修改 → 运行auto-sync.ps1 → 自动推送到GitHub → 网站自动更新
```

**时间线：**
1. 你在本地修改 `index.html`、`heartopia.css` 或 `heartopia.js`
2. 运行 `auto-sync.ps1` 脚本
3. 等待几秒钟，GitHub自动更新
4. 访问 `https://cant-1228.github.io/CAIT/` 查看更新

## 常见问题

### Q: 推送时提示需要登录？
A: 运行以下命令设置GitHub令牌：
```powershell
git remote set-url origin https://<你的令牌>@github.com/CANT-1228/CAIT.git
```

获取令牌：GitHub → Settings → Developer settings → Personal access tokens

### Q: 网站没有更新？
A: GitHub Pages需要1-3分钟更新，请稍等片刻后刷新页面

### Q: 如何查看修改历史？
A: 运行 `git log` 查看所有提交记录
