# Heartopia 攻略站

一个完整的 Heartopia 游戏攻略网站，为玩家提供全面的游戏指南。

## 功能特点

### 🎮 新手入门指南
- 游戏简介和基础玩法介绍
- 体力恢复和赚钱方法
- D.G.系统等级解锁表

### 🎣 钓鱼攻略
- 钓鱼基础技巧
- 最佳钓鱼时间和天气影响
- 热门鱼类速查表（地点、时间、天气、售价）

### 🌱 七大爱好攻略
- 钓鱼、园艺、烹饪、捕虫、观鸟、养猫、养狗

### 🍳 食谱大全
- 多种食谱分类（果酱、汤类、派类、沙拉、主菜、甜点）
- 动态筛选功能
- 详细的材料、等级、售价和能量信息

### 👥 NPC好感度攻略
- 8位NPC的喜好介绍
- 友情小贴士（每天对话、送礼技巧、生日送礼等）

### 🎬 视频教程
- Bilibili视频嵌入
- 图片攻略展示

### 📝 留言板
- 用户留言功能
- 本地存储保存留言
- 联系邮箱：2156200755@qq.com

### 🎁 兑换码
- 有效兑换码列表
- 已过期兑换码参考
- 一键复制功能
- 兑换方法说明

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（响应式布局）
- **JavaScript** - 交互功能
- **LocalStorage** - 数据存储

## 文件结构

```
heartopia-guide/
├── heartopia.html    # Heartopia攻略站主页面
├── heartopia.css     # Heartopia网站样式
├── heartopia.js      # Heartopia网站功能脚本
├── index.html        # 日常饮食网站页面
├── styles.css        # 日常饮食网站样式
├── script.js         # 日常饮食网站功能脚本
└── README.md         # 项目说明文档
```

## 使用方法

### 本地运行
1. 下载或克隆本仓库
2. 用浏览器打开 `heartopia.html` 文件
3. 开始使用！

### 部署到网站
你可以将这些文件部署到任何静态网站托管服务，如：
- GitHub Pages
- Netlify
- Vercel
- 或者你自己的服务器

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge
